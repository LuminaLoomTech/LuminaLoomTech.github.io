# 單頁滾動網站改造說明

## 改造內容

已將網站從多頁面路由改為單頁滾動設計，使用者可以在主頁往下滑動查看所有內容。

## 主要變更

### 1. 新增檔案

- **MainPage.tsx** - 整合所有頁面內容的主頁面
  - 包含：Home (首頁)、About (關於我們)、Services (服務項目)、Contact (聯絡表單)
  - 每個區塊都有獨立的 section id，可以透過錨點連結跳轉

- **MainPage.module.css** - 單頁設計的樣式
  - 響應式設計，支援桌面與手機版本
  - 每個 section 至少佔據一個螢幕高度 (min-height: 100vh)
  - 平滑滾動效果

### 2. 修改檔案

#### App.tsx
- 移除了 React Router 的多頁面路由配置
- 改為單一主頁面 (MainPage)
- 更新 menuItems 為錨點導航

#### Sidebar.tsx
- 從 `Link` (路由導航) 改為 `<a>` (錨點導航)
- 新增 `onClick` 回調函數支援
- 點擊選單項目時會平滑滾動到對應區塊

#### App.css
- 新增 `html { scroll-behavior: smooth; }` 支援平滑滾動

## 頁面結構

```
MainPage
├── #home - 首頁區塊
│   ├── Banner 輪播
│   ├── Intro 介紹
│   └── NewsSection 最新消息
├── #about - 關於我們
│   ├── 軟體部門介紹
│   ├── 遊戲部門介紹
│   └── 產品列表
├── #services - 服務項目
│   └── 6 個服務項目卡片
└── #contact - 聯絡表單
    └── 完整的聯絡表單 (使用 EmailJS)
```

## 導航方式

1. **側邊欄 (Sidebar)** - 點擊選單項目會平滑滾動到對應區塊
2. **錨點連結** - 可以直接使用 `#home`, `#about`, `#services`, `#contact`
3. **手動滾動** - 使用滑鼠滾輪或觸控手勢向下滾動
