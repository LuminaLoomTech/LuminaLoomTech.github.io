| 類別         | 功能               | 註解           |
| ---------- | ---------------- | ------------ |
| `max-w-md` | 最大寬度 28rem       | 限制表單太寬，保持美觀  |
| `w-full`   | 寬度 100%          | 讓輸入框或按鈕填滿容器  |
| `mx-auto`  | 左右自動置中           | 表單水平置中       |
| `p-4`      | padding 1rem     | 表單內部空間，不貼邊   |
| `m-4`      | margin 1rem      | 外部間距         |
| `mb-2`     | 下方 margin 0.5rem | 欄位間距         |
| `mb-4`     | 下方 margin 1rem   | 標題與欄位間距      |
| `flex`     | Flex 排版          | 水平排列元素       |
| `grid`     | Grid 排版          | 多欄位排版        |
| `gap-4`    | 子元素間距 1rem       | Flex/Grid 間距 |
